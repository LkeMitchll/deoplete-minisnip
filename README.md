# deoplete-minisnip

A Deoplete source for [vim-minisnip](https://github.com/joereynolds/vim-minisnip), the lightweight snippet manager.

## Installation

With vim-plug

```
Plug 'joereynolds/deoplete-minisnip'
```

You may need to `:UpdateRemotePlugins`
